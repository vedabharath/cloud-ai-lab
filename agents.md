## Conventions

### File Naming
- Python files: `snake_case.py`
- Java files: `PascalCase.java`
- Notebooks: `descriptive-name.ipynb`
- Markdown docs: `kebab-case.md` or `UPPERCASE.md` for root docs

### Code Style
- Python: Follow PEP 8
- Java: Follow Spring Boot conventions
- Include docstrings/comments for learning reference

### Documentation
- Each major folder should have its own `README.md`
- Include learnings, gotchas, and references in docs
